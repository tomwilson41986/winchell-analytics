# public/ assets

## winchell-silks.png (required brand asset)

`winchell-silks.png` is the **Winchell Thoroughbreds racing silks** logo. It is
referenced by:

- `index.html` — favicon
- `src/components/Logo.tsx` — header logo

**This file is provided by the project owner and must NOT be replaced with a
generated or substitute graphic.** Drop the supplied image in this folder as
`winchell-silks.png`. Files in `public/` are served at the site root, so it is
available at `/winchell-silks.png`.

Until the real image is added, the logo/favicon will show a broken-image
placeholder — that is expected.
